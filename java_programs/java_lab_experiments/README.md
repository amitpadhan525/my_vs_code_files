# Java Experiments

This project contains the Java files for the 10 experiments. These are compilable and runnable.

## Experiments List
1.  **HelloWorld**: Print Hello World.
2.  **DataTypesAndControl**: Demonstrate data types, operators, arrays, and control structures.
3.  **ConstructorDemo**: Demonstrate classes and constructors.
4.  **MethodOverloading**: Demonstrate method overloading.
5.  **InheritanceOverride**: Demonstrate inheritance and method overriding.
6.  **PackageDemo**: Demonstrate packages (includes `mypackage/MyClass.java`).
7.  **ExceptionHandling**: Demonstrate exception handling (Division by Zero).
8.  **MultithreadingDemo**: Demonstrate multithreading with 4 threads.
9.  **AppletStructure**: Demonstrate Applet structure and event handling.
10. **LayoutManagerDemo**: Demonstrate AWT/Swing Layout Managers.

## How to Run Java Files
To run any experiment (e.g., HelloWorld), run the following in this directory:
```bash
javac HelloWorld.java
java HelloWorld
```

For the package demo:
```bash
javac mypackage/MyClass.java
javac PackageDemo.java
java PackageDemo
```
